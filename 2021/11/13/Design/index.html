一、使用 text-align 属性创建视觉平衡

课程的这一部分侧重于应用视觉设计。第一组挑战建立在给定的卡片布局上，以展示一些核心原则。

文本通常是 Web 内容的很大一部分。CSS 有几个关于如何将其与text-align属性对齐的选项。

text-align: justify; 将文本间隔开，以便每行具有相等的宽度。

text-align: center; 居中文本

text-align: right; 右对齐文本

并且text-align: left;（默认）左对齐文本。

三、加粗与下划线

要使文本加粗，您可以使用strong标签。这通常用于引起对文本的注意并象征它很重要。使用strong标签，浏览器将 CSS 应用font-weight: bold;到元素。

要为文本加下划线，您可以使用u标签。这通常用于表示一段文本很重要，或者要记住一些东西。使用u标签，浏览器将 CSS 应用text-decoration: underline;到元素。
