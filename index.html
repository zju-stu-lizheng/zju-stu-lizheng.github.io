1、组帧的方法

(1)字符计数法

首部添加一个字节，表示该帧的长度（包括计数的这个字符），单位为字节。

(2)字符填充的首尾定界符法

首尾各添加一个FLAG标志，内容如果出现FLAG或者转义符ESC,则在这两个字符前添加一个转移符ESC.

(3)零比特填充的首尾标志法

首尾各添加一个0x7E字符，内容如果出现连续5个1自动后面补一个0.

(4)违规编码法

特殊地，针对曼彻斯特编码，我们可用 “ 高-高 ” , “ 低-低 ” 来定界帧的起始和终止。

3、流量控制

(1) 停等协议：

发送实体发送一个帧，接受实体接收处理完之后必须发回一个对于这个帧的确认表示自己同意接收下一个帧；发送方收到这个确认之后，才能发送下一个帧。所谓停等，接收方能够简单的通过停止发送确认的方式来阻止数据流的发送。

(2) 滑动窗口：

  • Go back N：发送窗口大小大于1，接收窗口大小为1；
  • 选择重传：发送窗口大小大于1，接收窗口大小大于1

考点：信道利用率

信道利用率：发送方在一个发送周期内，有效的发送数据所需的时间占整个发送周期的比率。

信道利用率=(L/C)/T信道利用率 = (L/C)/T 信道利用率=(L/C)/T

T：一个周期的时间

L:T时间内发送的数据量

C：发送方数据传输速率

File-System Structure

  • 文件结构
    • 逻辑储存模块
    • 相关信息的容器
  • 二级存储（disks）
  • 分层管理
  • FCB(文件控制块)：包含文件管理的信息

Layered file system

分层的目的：每层设计利用更低层的功能，创建新的功能，以用于更高层服务。

I/O control

包括设备驱动程序和中断处理程序，以在主内存和磁盘系统之间传输信息。设备驱动程序可以作为翻译器。

basic file system

只需向适当设备驱动程序发送通用命令，以读取和写入磁盘的物理块。每个物理块由磁盘的数字地址来标识（如，驱动器：1，磁道：3）。

也管理内存缓冲区和保存各种文件系统、目录和数据块的缓存。

file-organization module

知道文件及其逻辑块以及物理块。可以将逻辑块地址转为物理块地址，以供basic file system传输。

Logical file system

管理元数据信息。元数据包括文件系统的所有结构，而不包括实际数据（或文件内容）。逻辑文件系统Logical file system管理目录结构，以便根据给定文件名称为文件组织模块提供所需的信息。

它通过文件控制块来维护文件结构。文件控制块（PCB)包含有关文件的信息，包括所有者、权限、文件内容的位置等。

分配方法

连续分配

  • 要求：每个文件在磁盘上占有一组连续的块。

  • 文件的连续分配可以用首块的磁盘地址和连续的块数来定义。

  • 访问非常容易，支持顺序访问和直接访问。（对于直接访问一个文件的从块b开始的第i块，可以直接访问块b+i.

  • 一个难题是：为新文件找到空间。动态存储分配（容易造成外部碎片）

  • 另一个难题是：确定一个文件需要多少空间。

Extent-Based Systems

通过拓展来分配磁盘块

每个拓展是磁盘上的连续块

链表分配

  • 需要保留一个指针，指向下一个block

  • Simple -need only starting address
  • 没有浪费的空间
  • No random access,poor reliability
  • Mapping（第一个word放指向下一块的指针）

File-Allocation Table

mini-link:FAT每一项是一个word，存有指向下一块的位置。

使用内存块的指针链表代替磁盘的指针链表。

FAT表磁盘贮存，mount后加载到内存。

缺陷：如果一个pointer出错，可能导致后面的信息全部丢失

索引分配

  • Brings all pointer together into the index block
  • Logical views

缺点：需要多访问一次磁盘（索引块）

通常来说：会采用组合的模式：既有直接访问，也可以有多级访问

Free-Space Management

空闲块的管理

  • Bit map requires extra space
    • Example:
      • block size : 2122^{12}212bytes
      • disk size: 2302^{30}230bytes
      • n = 230/212=2182^{30}/2^{12} = 2^{18}230/212=218bits(32 K bytes)
  • Easy to get contiguous files
  • Linked list(free list)
    • Cannot get contiguous space easily
    • But basically can work (FAT)
    • No waste of space
  • Grouping - a modification of the Linked list
    • Addresses of the n free blocks are stored in the first block.
  • Counting
    • Address of the first free block and number n contiguous blocks.

Need to protect

  • 空闲链表的指针
  • bit map
    • Must be kept on disk
    • The copy in memory and disk may differ
    • Cannot allow for block[i] to have a situation where bit[i] = 1 in memory and bit[i] = 0 on disk
  • Solution:
    • 先给硬盘上的 bit[i]置1
    • 再释放block[i]
    • 最后改内存上的bit[i]

实现Phong光照模型

一、光照模型简介

在 OpenGL 中，常用的一个光照模型被称为冯氏光照模型(Phong Lighting Model)。冯氏光照模型的主要结构由3个分量组成：环境(Ambient)、漫反射(Diffuse)和镜面(Specular)光照。下面这张图展示了这些光照分量看起来的样子：

环境光照(Ambient Lighting)：为了模拟真实世界中，即便在黑暗的情况下，物体也不会永远是黑暗的这样一个事实，我们使用一个常量始终给物体一些颜色。

漫反射光照(Diffuse Lighting)：模拟光源对物体的方向性影响(Directional Impact)。它是冯氏光照模型中视觉上最显著的分量。物体的某一部分越是正对着光源，它就会越亮。

镜面光照(Specular Lighting)：模拟有光泽物体上面出现的亮点。镜面光照的颜色相比于物体的颜色会更倾向于光的颜色。

Graph Attention Networks

参考文献 ：《Graph Attention Networks》

Introduction

针对图结构数据，本文提出了GAT（graph attention networks)网络。该网络使用self-attention层解决了之前基于图卷积的模型所存在的问题。在GAT中，图中的每个节点可以根据邻节点的特征，为其分配不同的权值。GAT的另一个优点在于，无需使用预先构建好的图。因此GAT可以解决一些基于谱的图神经网络中所具有的问题。

node2vec

node2vec : Scalable Feature Learning for Networks

1. Introduction：

  1. 介绍网络分析的任务

    • 网络节点的分类(node classification task):将网络中的节点进行聚类，哪些节点有着相同的属性，就分为一类。
    • 链接预测(Link prediction)：预测网络中哪些顶点有着潜在的关联。
    • …

  2. 网络嵌入问题

    完成上述任务的前提是解决网络嵌入问题，以前的手工提取特征的方式有着诸多不足。

    • 解优化函数的方式学习网络的表示特征（learn feature representations by solving an optimization problem)：面临计算效率和准确度的平衡问题，无法兼顾两者。使用传统的降维方法如PCA（主成分分析）也确实有一定效果，但是会涉及矩阵分解，运算量大准确率也不高。
    • 本文尝试设计出既能保持节点邻居特性而且又特别容易训练的模型。我们发现很多节点在网络中往往有一些类似的结构特征。一种结构特征是很多节点会聚集在一起，内部的连接远比外部的连接多，我们称之为社区（community）；另一种结构特征是网络中两个可能相聚很远的点在边的连接上有着相似的特征。
    • 如下图，u,s1,s2,s4,s4u,s_1,s_2,s_4,s_4u,s1​,s2​,s4​,s4​属于一个社区，u,s6u,s_6u,s6​在结构上有着相似的特征。

要设计的网络表示学习算法的目标必须满足这两点：

  • 同一个社区内节点表示相似。
  • 拥有类似结构特征的节点表示相似。

总之本文的贡献是：提出了一个有效的，可扩展的表示学习算法，可以体现网络特征和节点邻居特征。

2. Related Works :

之前的工作存在以下问题：

  • 特征需要依赖人手工定义，不是某个领域内的专业人士无法从事该工作。
  • 一些无监督学习（Unsupervised feature learning）中的降维方法拿来使用，但是这些方法计算效率低，准确度也不够，而且还不能反映出网络的结构特征。

自然语言处理（natural language processing）表征学习的进展为本文的作者提供了思路。与deepwalk相似，这里也用skip-gram模型来解决网络表示学习的问题。
