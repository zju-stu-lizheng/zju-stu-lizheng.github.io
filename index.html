Graph Attention Networks

参考文献 ：《Graph Attention Networks》

Introduction

针对图结构数据，本文提出了GAT（graph attention networks)网络。该网络使用self-attention层解决了之前基于图卷积的模型所存在的问题。在GAT中，图中的每个节点可以根据邻节点的特征，为其分配不同的权值。GAT的另一个优点在于，无需使用预先构建好的图。因此GAT可以解决一些基于谱的图神经网络中所具有的问题。

GAT architecture

Graph attentional layer

首先来介绍单个的graph attentional layer。

  • 单个的 graph attentional layer 的输入是一个节点特征向量集：

h=h1⃗,h2⃗,...,hN⃗,hi⃗∈RFh = {\vec{h_1},\vec{h_2},...,\vec{h_N}},\vec{h_i} \in {\mathcal R}^F h=h1​​,h2​​,...,hN​​,hi​​∈RF

其中，NNN表示节点集中节点的个数，FFF表示相应的特征向量维度。

  • 每一层的输出是一个新的节点特征向量集：

h′=h1′⃗,h2′⃗,...,hN′⃗,hi′⃗∈RF′h' = {\vec{h'_1},\vec{h'_2},...,\vec{h'_N}},\vec{h'_i} \in {\mathcal R}^{F'} h′=h1′​​,h2′​​,...,hN′​​,hi′​​∈RF′

其中，F′F'F′ 表示新的节点特征向量维度（可以不等于FFF)。

具体来说，graph attentional layer首先根据输入的节点特征向量集，进行self-attention处理：

eij=a(Whi⃗,Whj⃗)e_{ij} = a(W\vec{h_i},W\vec{h_j}) eij​=a(Whi​​,Whj​​)

其中，a是一个 RF′×RF′→R{\mathcal R}^{F'} \times {\mathcal R}^{F'} \rightarrow {\mathcal R}RF′×RF′→R的映射，W∈RF′×FW \in {\mathcal R}^{F'\times F}W∈RF′×F是一个权值矩阵。一般来说，self-attention会将注意力分配到图中所有的节点上，这种做法显然会丢失结构信息。为了解决这个问题，本文使用了masked attention的方式——仅将注意力分配到点iii的邻节点集上，即j∈Nij \in \mathcal N_ij∈Ni​（在本文中，节点i也是Ni\mathcal N_iNi​的一部分）：

GCN

论文：Semi-Supervised Classification with Graph Convolutional Networks
作者：Thomas N. Kipf, Max Welling
来源：ICLR 2017

0. Abstract：

semi-GCN是一篇经典的GCN论文，作者提出了一种简单且有效GCN框架，使用切比雪夫一阶展开近似谱卷积，使每一个卷积层仅处理一阶邻域信息，然后通过分层传播规则叠加一个个卷积层达到多阶邻域信息传播。

1. 基于图的半监督学习:

传统方法基于“相连节点应该相似，具有相同标签”的假设，将基于图的正则项显式加入到损失函数中进行学习。

但是，这个假设太严格了，极大限制了模型能力，因为节点相连不一定都是相似，但是会包含附加信息。

node2vec

node2vec : Scalable Feature Learning for Networks

1. Introduction：

  1. 介绍网络分析的任务

    • 网络节点的分类(node classification task):将网络中的节点进行聚类，哪些节点有着相同的属性，就分为一类。
    • 链接预测(Link prediction)：预测网络中哪些顶点有着潜在的关联。
    • …

  2. 网络嵌入问题

    完成上述任务的前提是解决网络嵌入问题，以前的手工提取特征的方式有着诸多不足。

    • 解优化函数的方式学习网络的表示特征（learn feature representations by solving an optimization problem)：面临计算效率和准确度的平衡问题，无法兼顾两者。使用传统的降维方法如PCA（主成分分析）也确实有一定效果，但是会涉及矩阵分解，运算量大准确率也不高。
    • 本文尝试设计出既能保持节点邻居特性而且又特别容易训练的模型。我们发现很多节点在网络中往往有一些类似的结构特征。一种结构特征是很多节点会聚集在一起，内部的连接远比外部的连接多，我们称之为社区（community）；另一种结构特征是网络中两个可能相聚很远的点在边的连接上有着相似的特征。
    • 如下图，u,s1,s2,s4,s4u,s_1,s_2,s_4,s_4u,s1​,s2​,s4​,s4​属于一个社区，u,s6u,s_6u,s6​在结构上有着相似的特征。

要设计的网络表示学习算法的目标必须满足这两点：

  • 同一个社区内节点表示相似。
  • 拥有类似结构特征的节点表示相似。

总之本文的贡献是：提出了一个有效的，可扩展的表示学习算法，可以体现网络特征和节点邻居特征。

2. Related Works :

之前的工作存在以下问题：

  • 特征需要依赖人手工定义，不是某个领域内的专业人士无法从事该工作。
  • 一些无监督学习（Unsupervised feature learning）中的降维方法拿来使用，但是这些方法计算效率低，准确度也不够，而且还不能反映出网络的结构特征。

自然语言处理（natural language processing）表征学习的进展为本文的作者提供了思路。与deepwalk相似，这里也用skip-gram模型来解决网络表示学习的问题。
