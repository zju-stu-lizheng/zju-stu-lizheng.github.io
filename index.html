GCN

论文：Semi-Supervised Classification with Graph Convolutional Networks
作者：Thomas N. Kipf, Max Welling
来源：ICLR 2017

0. Abstract：

semi-GCN是一篇经典的GCN论文，作者提出了一种简单且有效GCN框架，使用切比雪夫一阶展开近似谱卷积，使每一个卷积层仅处理一阶邻域信息，然后通过分层传播规则叠加一个个卷积层达到多阶邻域信息传播。

1. 基于图的半监督学习:

传统方法基于“相连节点应该相似，具有相同标签”的假设，将基于图的正则项显式加入到损失函数中进行学习。

但是，这个假设太严格了，极大限制了模型能力，因为节点相连不一定都是相似，但是会包含附加信息。

(7) utils

LIL（Row-Based Linked List Format）-基于行的链表格式

稀疏矩阵转化成两个链表data和rows：

  • 列表.data: data[k]是行k中的非零元素的列表。如果该行中的所有元素都为0，则它包含一个空列表。
  • 列表.rows: 是在位置k包含了在行k中的非零元素列索引列表。
载入数据的维度（以Cora数据集为例）
  • adj(邻接矩阵)：由于比较稀疏，邻接矩阵格式是LIL的，并且shape为(2708, 2708)
  • features（特征矩阵）：每个节点的特征向量也是稀疏的，也用LIL格式存储，features.shape: (2708, 1433)
  • labels：ally, ty数据集叠加构成，labels.shape:(2708, 7)
  • train_mask, val_mask, test_mask：shaped都为(2708, )的向量，但是train_mask中的[0,140)范围的是True，其余是False；val_mask中范围为(140, 640]范围为True，其余的是False；test_mask中范围为[1708,2707]范围是True，其余的是False
  • y_train, y_val, y_test：shape都是(2708, 7) 。y_train的值为对应与labels中train_mask为True的行，其余全是0；y_val的值为对应与labels中val_mask为True的行，其余全是0；y_test的值为对应与labels中test_mask为True的行，其余全是0
  • 特征矩阵进行归一化并返回一个格式为(coords, values, shape)的元组
  • 将邻接矩阵加上自环以后，对称归一化，并存储为COO模式，最后返回格式为(coords, values, shape)的元组

Graph Attention Networks

参考文献 ：《Graph Attention Networks》

Introduction

针对图结构数据，本文提出了GAT（graph attention networks)网络。该网络使用self-attention层解决了之前基于图卷积的模型所存在的问题。在GAT中，图中的每个节点可以根据邻节点的特征，为其分配不同的权值。GAT的另一个优点在于，无需使用预先构建好的图。因此GAT可以解决一些基于谱的图神经网络中所具有的问题。

GAT architecture

Graph attentional layer

首先来介绍单个的graph attentional layer。

  • 单个的 graph attentional layer 的输入是一个节点特征向量集：

h=h1⃗,h2⃗,...,hN⃗,hi⃗∈RFh = {\vec{h_1},\vec{h_2},...,\vec{h_N}},\vec{h_i} \in {\mathcal R}^F h=h1​​,h2​​,...,hN​​,hi​​∈RF

其中，NNN表示节点集中节点的个数，FFF表示相应的特征向量维度。

  • 每一层的输出是一个新的节点特征向量集：

h′=h1′⃗,h2′⃗,...,hN′⃗,hi′⃗∈RF′h' = {\vec{h'_1},\vec{h'_2},...,\vec{h'_N}},\vec{h'_i} \in {\mathcal R}^{F'} h′=h1′​​,h2′​​,...,hN′​​,hi′​​∈RF′

其中，F′F'F′ 表示新的节点特征向量维度（可以不等于FFF)。

  • 一个graph attention layer的结构如下图所示：

具体来说，graph attentional layer首先根据输入的节点特征向量集，进行self-attention处理：

eij=a(Whi⃗,Whj⃗)e_{ij} = a(W\vec{h_i},W\vec{h_j}) eij​=a(Whi​​,Whj​​)

其中，a是一个 RF′×RF′→R{\mathcal R}^{F'} \times {\mathcal R}^{F'} \rightarrow {\mathcal R}RF′×RF′→R的映射，W∈RF′×FW \in {\mathcal R}^{F'\times F}W∈RF′×F是一个权值矩阵。一般来说，self-attention会将注意力分配到图中所有的节点上，这种做法显然会丢失结构信息。为了解决这个问题，本文使用了masked attention的方式——仅将注意力分配到点iii的邻节点集上，即j∈Nij \in \mathcal N_ij∈Ni​（在本文中，节点i也是Ni\mathcal N_iNi​的一部分）：

αij=softmaxj(eij)=exp(eij)∑k∈Niexp(eik)\alpha_{ij} = softmax_j(e_{ij}) = \frac{exp(e_{ij})}{\sum_{k\in \mathcal N_i}exp(e_{ik})} αij​=softmaxj​(eij​)=∑k∈Ni​​exp(eik​)exp(eij​)​

在本文中，aaa使用单层的前馈神经网络实现。总的计算过程为：

αij=exp(LeakyReLU(a⃗T[Whi⃗∣∣Whj⃗]))∑k∈Niexp(LeakyReLU(a⃗T[Whi⃗∣∣Whk⃗]))\alpha_{ij} = \frac{exp(LeakyReLU(\vec{a}^T[W\vec{h_i}||W\vec{h_j}]))}{\sum_{k\in \mathcal N_i}exp(LeakyReLU(\vec{a}^T[W\vec{h_i}||W\vec{h_k}]))} αij​=∑k∈Ni​​exp(LeakyReLU(aT[Whi​​∣∣Whk​​]))exp(LeakyReLU(aT[Whi​​∣∣Whj​​]))​

  • ps：Leaky ReLU函数图像如下：（Leaky ReLUs allow a small, positive gradient when the unit is not active）

其中，a⃗T∈R2F′\vec{a}^T \in {\mathcal R}^{2F'}aT∈R2F′为前馈神经网络a的参数，Leaky ReLU为前馈神经网络的激活函数，此时即可得到hi′⃗\vec{h'_i}hi′​​了：

hi′⃗=σ(∑j∈NiαijWhj⃗)\vec{h'_i} = \sigma(\sum_{j\in {\mathcal N_i}}\alpha_{ij}W\vec{h_j}) hi′​​=σ(j∈Ni​∑​αij​Whj​​)

为了提高模型的拟合能力，在本文中还引入了多抽头的self-attention（如图一右侧部分。与《Attention is All You Need》一致），即同时使用多个 WkW^kWk 计算 self-attention, 然后将各个 WkW^kWk 计算得到的结果合并（连接或求和)：

hi′⃗=∣∣k=1Kσ(∑j∈NiαijWhj⃗)\vec{h'_i} = {||}^K_{k=1} \sigma(\sum_{j\in {\mathcal N_i}}\alpha_{ij}W\vec{h_j}) hi′​​=∣∣k=1K​σ(j∈Ni​∑​αij​Whj​​)

其中，||表示连接，aijka_{ij}^kaijk​与WkW^kWk表示第k个抽头得到的结果。由于Wk∈RF′×FW^k \in {\mathcal R}^{F'\times F}Wk∈RF′×F，因此这里的hi′⃗∈RKF′\vec{h'_i} \in {\mathcal R}^{KF'}hi′​​∈RKF′。同样可以采取求和的方式来得到hi′⃗\vec{h'_i}hi′​​：

hi′⃗=σ(1K∑k=1K∑j∈NiαijWhj⃗)\vec{h'_i} = \sigma(\frac{1}{K}\sum_{k=1}^K \sum_{j\in {\mathcal N_i}}\alpha_{ij}W\vec{h_j}) hi′​​=σ(K1​k=1∑K​j∈Ni​∑​αij​Whj​​)
